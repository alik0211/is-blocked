# Is blocked?

Tiny library based on fetch to check sites for blocking.

## Usage

```js
isBlocked('https://t.me').then(() => {
  // Do something if the site is blocked
});
```

## Install

### Module

Using npm:

```
npm i is-blocked
```

Or yarn:

```
yarn add is-blocked
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/is-blocked@2.0.0/dist/is-blocked.min.js"></script>
```
